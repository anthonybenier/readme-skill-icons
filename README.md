Generate dynamic, beautiful SVG skill icons for your GitHub Profile README.

## New feature: Badge builder

You can now configure and add badges from shields.io ! 
Create **Custom badges** with your own text or dynamic **GitHub badges** to display live repository stats (Stars, Forks, License, etc.). 

![Image](https://github.com/anthonybenier/readme-icons/blob/main/public/readme_assets/badgeBuilder.png?raw=true)


## Credits
This project uses **[Simple Icons](https://simpleicons.org/)** for technology logos and **[Shields.io](https://shields.io/)** for badge generation.
**Missing an icon?** Please check if it exists on Simple Icons first. If not, open an issue/PR there. We update our dependencies regularly.

## For Developers

1. **Clone & Install**
   ```bash
   git clone https://github.com/anthonybenier/readme-icons.git && cd readme-icons && npm install
   ```

2. **Run**
   ```bash
   npm run dev
   ```
